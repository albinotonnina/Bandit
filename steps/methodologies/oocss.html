OOCSS in a nutshell

Merits

  • Abstract styles into reusable LEGO classes
  • Single responsibility principle (SRP) approach
  • Separation of concerns principle
  • Lower specificity, equally-specific class selectors
  • Reduced duplication
  • Smaller file size
  • Concept of "performance freebies"
  • Fast prototyping
