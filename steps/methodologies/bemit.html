Augmented BEM naming convention

BEMIT in a nutshell

Namespaces add another layer of rich meaning

from 
                    


  <img class="media__img user__photo avatar"/>
  
...
to 
                    


  <img class="o-media__img c-user__photo c-avatar"/>
  
...

c- for Components, o- for Objects, u- for Utilities, and is-/has- for States

BEMIT in a nutshell

Responsive Suffixes

from 
                    


  <img class="o-media__img c-user__photo c-avatar"/>
  
...
to 
                    


  <img class="o-media__img@md c-user__photo c-avatar"/>
  
...

"this class when at this size"

BEMIT in a nutshell

Responsive Suffixes examples

                    
/* escape the @ symbol in your CSS file */

@media print {

  .u-hidden\@print {
    display: none;
  }

}

                    
                
                    
.u-hidden\@print{
    /* hide things when in print context. */
}

.u-1/4\@lg{
    /* a quarter width in the large breakpoint. */
}

.o-layout\@md{
    /* a layout object in the medium breakpoint. */
}
