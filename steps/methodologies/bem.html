BEM in a nutshell

Merits

  • Abstraction of modules into BEM modules (blocks)
  • Single responsibility principle (SRP) approach
  • Strict rules
  • Lower specificity, equally-specific class selectors
  • Fast prototypes

BEM in a nutshell

Faults

  • Bloats the HTML 
    

    Look! Classes!
  • Bad DX, poor readability
  • Easy to abuse descendants selectors 
    
.classes__about__five__levels__deep{}
.block__element__element{}
  • Frequent changes on HTML
  • Learning curve
  • Difficult abstractions sometimes
